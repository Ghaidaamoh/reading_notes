# 1. logical operators:

**logical operators in JavaScript:**
- OR
- AND
- NOT

**(OR)tabels:**
- true OR true/  true
- false OR true/ true
- true OR false/  true
- false OR false/ false

**(and)tabels:**
- true And true/   true
- false And true/  false
- true And false/ false
- false And false/ false

 **(NOT) result = !value**
